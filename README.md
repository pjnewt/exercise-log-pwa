# 6-Week Exercise Log PWA

A progressive web app to track your 6-week exercise program with detailed session information.

## Features

- Track 42 sessions across 6 weeks
- Detailed exercise descriptions
- Offline functionality
- Installable as standalone app
- Responsive design for mobile devices

## Installation

1. Visit the GitHub Pages URL
2. Click the menu (three dots) → "Add to Home screen"
3. Use as a standalone app on your Android device

## Usage

1. Complete sessions by entering dates
2. Add notes for each session
3. View completion statistics
4. Works offline after first load
